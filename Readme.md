# raw-solarized

(Solarized scheme)[https://github.com/altercation/ethanschoonover.com/blob/master/projects/solarized/README.md] color values in json format.

## Format

```js
{
  "base03": {
    "hex": "#002b36",
    "16/8": "8/4",
    "termcol": "brblack",
    "xterm": {"code": 234, "hex": "#1c1c1c"},
    "lab": {"l": 15, "a": -12, "b": -12},
    "rgb": {"r": 0, "g": 43, "b": 54},
    "hsb": {"h": 193, "s": 100, "b": 21}
  },
  ...  
}
```
